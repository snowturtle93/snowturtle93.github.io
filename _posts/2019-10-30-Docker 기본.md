1. Docker 서비스 등록
<pre><code>
$sudo systemctl start docker

$sudo systemctl enable docker

systemctl status docker.service
</code></pre>

2. 현재 로그인된 사용자에게 권한줌(sudo없이 도커 사용가능).
<pre><code>
sudo usermod -aG docker $USER
</code></pre>
